# Change Log

All notable changes to the "StatusMark" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-11-04

### Changed
- Added demo GIF to README for better visualization
- Cleaned up README by removing placeholder sections
- Improved documentation structure

## [1.0.0] - 2025-11-04

### Added
- Initial release of StatusMark
- Display custom project name and emoji in VS Code status bar
- Live preview while configuring label settings
- Color picker with curated palette for label text color
- Color picker with curated palette for status bar background
- Automatic contrasting text color calculation for status bar
- Three commands:
  - `StatusMark: Configure` - Complete label and status bar setup
  - `StatusMark: Configure Status Bar Color` - Quick status bar color adjustment
  - `StatusMark: Reset Workspace Settings` - Reset all settings to defaults
- Support for custom colors via hexadecimal input
- Theme default options for both badge and status bar colors
- Workspace-level configuration persistence
- Real-time badge updates when configuration changes
